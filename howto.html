HTML

Hyper Text Markup Language(HTML) is the main language that is used in creating this website. The code above at the code that is used to make my navbar is not HTML. Instead HTML code looks like this:


Doesnt look like much right? Yep, its not at all. Lets go over some of the different pieces and tags that makes this up to prove just how simple it is. Lets start with how we can see which ones are tags. A tag starts with <> and the end of the tag looks like this(notice the slash) </>

The letters that go inside of these arrows is all dependent on what you want the tag to represent. There are several very important and common tags that you will use; <p> For Paragraphs </p> , <h1> For Beginning Headings or Large Headings </h1> , <h2> For slightly smaller Headers </h2>

These tags often make up the main portion of your web page, but there are many other useful tags. For instance, notice how I am able to insert these useful images, this is possible through html. The code for this does not have a end tag and it looks like < img src= > Lets break this down; img is pretty simple because all it does is let the computer know that there is an image to be displayed, the src stands for source and lets the computer know where the image is located. And boom, thats it for the image tag so now you can insert images anywhere.

And thats pretty much all you need to know to make your website in the regards of html. This language may be useful however there are still many limitations. For one, this language is naturally not meant for designing websites, instead this language is meant to be spoken over the internet and understood as fast as possible, this being said it is limited to how much you can customize it. Notice as I explain in my CSS section.
